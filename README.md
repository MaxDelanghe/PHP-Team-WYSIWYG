# MY_WYSIWYG

Plugin WYSIWYG (what you see is what you get) for add buttons and modifie your textarea

## Installation

Add the script to your HTML

```js
<script src='my_wysiwyg.js'></script> 
```


## FontAwesome and css :

Get buttons icon with font-awesome

``` 
https://cdnjs.cloudflare.com/ajax/libs/font-awesome/5.15.2/css/all.min.css
```

Add the following lines to your css :

```css

.buttons {
  display: grid;
  grid-template-columns: 1fr 1fr;
}

.editor {
  width: 100%;
  height: auto;
  border-right: 1px solid black;
  background-color: white;
  word-break: break-all;
  overflow: scroll;
}

```
